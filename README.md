# Cyber-Threat-Intelligence-Analysis-of-Lazarus-Group-Using-MISP-
This project demonstrates the complete Cyber Threat Intelligence (CTI) lifecycle using the Malware  Information Sharing Platform (MISP). The objective of this project is to simulate a real-world  investigation of a known Advanced Persistent Threat (APT) group by creating a threat event, adding  Indicators of Compromise (IOCs)
